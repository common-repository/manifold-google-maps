The most effortless, valuable and effective google map plugin! Effectively make a boundless number of Google Maps and markers. 

We have given numerous alternatives to customize your map and marker completely.

*   Possibility to make an unlimited number of locations on each map
*   Fits in any screen
*   Mobile friendly
*   Display tooltip when you click a location icon
*   Customize tooltip

*   Easily compatible with import and export tools
*   Easily set different images to markers pins

*   Easy guidance for the setting up maps
*   Wordpress coding standard
*   Any theme compatible
*   Light weight

== Installation ==
1. Upload the entire manifold-google-maps folder(unzip) to the /wp-content/plugins/ directory or use dashboard (Plugins > Add new) and search manifold google maps
2. Activate the plugin through the /Plugins/ menu in WordPress.
3. Add the shortcode [manifold-google-maps cat=”YOUR_MAP_CATEGORY_SLUG”] where you want to display the map.
4. IMPORTANT : Add your Google API key in setting page Map Multi Marker

== Frequently Asked Questions ==
= <span id="faq">After installing the plugin what do we need to do?</span>  =

Once you've installed and activated the plugin, a Maps menu should appear in your left navigation panel within your WP-ADMIN section. Click on the Settings link and follow the on screen instructions.

= How do I put a Google map on my page/post? =

1. Create Google Map Key - add that key in plugin settings
2. Create Maps to create shortcode
3. Add markers and assign marker to category
4. Add the shortcode [manifold-google-maps cat=\"YOUR_MAP_CATEGORY_SLUG\"] where you want to display the map.
